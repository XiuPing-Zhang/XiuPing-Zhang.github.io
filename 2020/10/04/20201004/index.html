10.简单理解 ajax、axios

原生 ajax 请求：

1.创建对象：var ajax = new XMLHttpRequest()

2.设置请求：ajax.open(‘GET’,url,true)

3.请求头：ajax.setRequestHeader(“content-type”,”application/x-www-form-urlencoder”)

4.发送请求：ajax.send(null)

5.接受数据：ajax.onreadystatechange = function(obj){

​ if(obj.status===200){

​ ….

​ }

​ }

axios：基于 promise，可以拦截请求和响应，最底层也是基于 XMLHttpRequest，然后用 promise 封装

在浏览器中发送 XMLHttpRequest 请求，在 node 环境中发送 http 请求

请求拦截：多应用于项目当中，对 axios 进行封装，设置诸如 baseUrl 以及请求头等

axios.interceptors.request.use(function(config){

​ //发送请求前做的一些处理，诸如设置 token，baseUrl 等

​ return config；

},function(error){

​ return Promise.reject(error)
